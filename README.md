# zig-gp32

## Getting started

- [zig](https://ziglang.org/download/)
- [devkitPro](https://devkitpro.org/wiki/Getting_Started)

```
pacman -S gp32-dev
git clone https://github.com/zig-homebrew/zig-gp32
cd zig-gp32/
zig build # then run zig-out/zig-gp32.fxe with geepee32
```
